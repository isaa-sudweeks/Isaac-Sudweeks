Personal Project

Wave Machine

Concept device that creates a surfable river wave using fluid mechanics and portable infrastructure.

Fluid Dynamics Concept Design Prototyping

Overview

The wave machine explores how portable structures can shape a river flow into a stable standing wave. The concept was validated with a proof-of-concept model and peer feedback.

What I built

  • Scaled model to test geometry and flow behavior.
  • Concept sketches and CFD-inspired calculations.
  • Stakeholder review to refine safety assumptions.

What I learned

  • How flow rate and channel geometry interact.
  • Building a concept pitch with measurable assumptions.
  • Iterating designs based on feedback loops.
