Chemistry Testing Box

Continuous monitoring rig that measures atmospheric compounds and automates sample handling for lab experiments.

Instrumentation Data Acquisition Controls

Overview

The testing box combines sensors, actuators, and a GUI to track concentration changes in real time. It supports long experiments and reduces manual logging for lab teams.

What I built

  • Electronics integration for sensors, DACs, and LCDs.
  • GUI for live monitoring and control parameters.
  • Power conversion and solenoid actuation logic.

What I learned

  • Designing for long-term stability and calibration.
  • Data visualization choices that help researchers act faster.
  • Balancing sensor accuracy with system complexity.
