Work Project

Servo Testing Rig

A modular test platform for rope-driven servos that runs continuous durability cycles with automated logging.

Test Rig Arduino Reliability

Overview

This rig validates servo designs under extended duty cycles. I built the interface circuitry and control logic to safely drive PLC-style servo drivers while capturing cycle data.

What I built

  • Transistor interface board between Arduino and PLC inputs.
  • Fixture design for repeatable loading and alignment.
  • Long-run monitoring setup for thousands of hours.

What I learned

  • How to harden electronics for 24/7 operation.
  • Importance of modular test fixtures for rapid swaps.
  • Capturing failure modes with consistent logging.
