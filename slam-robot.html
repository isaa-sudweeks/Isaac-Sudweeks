Personal Project

SLAM Robot

Built an autonomous robot that maps and navigates unknown environments using a basic SLAM pipeline.

Robotics ROS Perception

Overview

This project focused on perception and navigation with a tracked robot platform. I integrated depth sensing, tuned SLAM parameters, and validated path planning in indoor spaces.

What I built

  • Jetson Nano compute stack with Intel RealSense depth camera.
  • ROS navigation pipeline with mapping and localization nodes.
  • Field tests to validate map quality and localization drift.

What I learned

  • Tradeoffs between mapping speed and accuracy.
  • Sensor placement effects on depth coverage.
  • How to diagnose SLAM failures in real time.
